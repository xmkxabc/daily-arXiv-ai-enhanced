使用说明

新功能使用指南
  • 深色模式: 点击导航栏右侧的月亮/太阳图标切换主题
  • 搜索建议: 在搜索框中输入2个字符以上即可看到建议
  • 论文评分: 点击论文卡片中的星星进行评分
  • 添加标签: 点击论文卡片中的"标签"按钮管理自定义标签
  • 添加笔记: 点击论文卡片中的"笔记"按钮添加个人笔记
  • 键盘快捷键: 在搜索框中使用上下箭头导航建议，Enter确认，Esc关闭
